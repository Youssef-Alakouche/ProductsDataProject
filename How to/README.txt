
Prerequisite : 
    - node installed on your machine
    - command line (terminal(mac or linux), Power Shell (window));

Github
- clone the project from github in any 

Run the project 
- navigate from terminal (or Power Shell) to project directory
- run "npm install" (to install dependencies that the project needs in order to work)
- create a copy of "Master file" (the original file).
- rename this file to "Data.xlsx"
- then run "node main.js"
